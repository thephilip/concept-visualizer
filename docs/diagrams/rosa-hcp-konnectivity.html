Konnectivity in ROSA HCP
How the control plane tunnels traffic to worker nodes when they live in separate networks.
⚠ ROSA HCP / HyperShift. In HyperShift the control plane runs in a management cluster on a different network from the worker nodes. Konnectivity provides a reverse tunnel so the API server, operators, and proxy sidecars can reach kubelets, services, and endpoints on the data plane. This architecture does not apply to ROSA Classic or self-managed OCP.
